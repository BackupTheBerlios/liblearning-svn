Near-duplicate detection and its application to spam filtering

Abstract We present a method for the problem of near-duplicate detection. Our approach scales well with the number of documents and is based on fingerprints which are computed over a so-called canonical form of the documents. Each fingerprint requires 64 bit. For a document $d$ of length $|d|$ the expexted number of fingerprints that needs to be stored is only $O(\log |d|)$ which is an improvement compared to other approaches.
