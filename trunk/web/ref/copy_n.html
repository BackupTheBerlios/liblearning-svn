copy_n<int N>

Description

Copies N elments without using loops. For small N this method also known as loop unroll may improve performance at run time.

Example

int n[] = {1, 2, 3};
copy_n<2>::assign(&n[1], &n[0]);
Copies the element 2 from the second to the first position and the element 3 from the third to the second position.
