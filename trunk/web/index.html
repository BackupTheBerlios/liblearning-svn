liblearning

C++ template library for Information Retrieval, Machine Learning, Data Mining, Feature Extraction

This is the home of liblearning, a C++ template library for information retrieval, machine learning, data mining and feature extraction. Our goal is to provide an easy to use framework for rapid prototype implementation to support scientific research on research areas mentioned above.
