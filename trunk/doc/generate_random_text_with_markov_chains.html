$$TITLE$$

The following example demonstrates how to create random text from a set of plain text documents. The documents are read and the contents are used as examples to train the Markov chain. For each two consecutive characters xy the following character z is observed and the number of occurrences is recorded. If all documents have been read the Markov chain is created by computing the transition probabilities, i.e. the probability that given a state xy the next state is yz. @see markov_chain, file_reader, directory_iterator
